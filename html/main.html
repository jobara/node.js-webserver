Fluid Integration Examples

The following examples illustrate more than one Fluid component, integrated into real-world settings.

uPortal Instance

  • Demo uPortal site

This is a live instance of uPortal, built from their trunk nightly. It incorporates the following Infusion features:

  • Fluid Skinning System The uPortal interface is styled using the Fluid Skinning System (FSS).
  • Layout Reorderer The portlets are reorderable using the Infusion Layout Reorderer.
  • Pager If you log in as "admin" and choose the "Portlet Manager" in the right sidebar, the list of registered portlets is paged using the Infusion Pager.
  • As well, if you choose "Add Content" in the right sidebar, you can add the "Javascript Demos" portlet (in the Applications section) which demonstrates the List Reorderer and the Pager. You can also add the "uPortal Powered Sites" portlet (in the uPortal section) which uses the Pager.

Sakai Mock-up

The Sakai Mock-up incorporates

  • an ideal example of dynamic restyling using the Fluid Skinning System and UI Options. Click the "Edit Appearance" button at the bottom-left of the page.

uPortal 3 Mock-up

The uPortal 3 Mock-up incorporates:

  • two versions of the Reorderer (the portlets are reorderable, and the images in the Image Gallery are reorderable)
  • an Inline Edit (the "Default folder behavior" text in the Bookmarks portlet is editable)
  • dynamic restyling of markup not using the Fluid Skinning System (use the "Customize My Portal" links in the upper right corner)
